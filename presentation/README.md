# מצגת אינטראקטיבית על רשתות נוירונים

הקבצים בתיקייה `presentation` בונים דף אחד אינטראקטיבי בעברית שמבוסס על החומר שבתיקיית `docs`.

מה יש בפנים:

- סיפור ב-6 פרקים שממשיך את הזרימה של `docs/video/interactive`
- דמו ציור ספרות בזמן אמת
- preprocessing ל-28x28
- הרצת מודל MNIST מקומי מתוך `data/mnist_model.js`
- אנימציית זרימת מידע ברשת
- פירוק אינטואיטיבי לפיצ'רים
- המחשת `solution space` ו-XOR

כדי לפתוח:

1. פתח את `index.html` בדפדפן.
2. צייר ספרה או לחץ על אחת הדוגמאות 0-9.
3. גלול בין הפרקים או השתמש בניווט העליון.

אם תרצה לעדכן את קובץ המודל:

- הקובץ `build_mnist_assets.py` מאמן מודל חדש וכותב את `data/mnist_model.js`.
